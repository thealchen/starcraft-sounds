# StarCraft Sounds for Claude Code

Plays a random StarCraft sound every time you submit a prompt.



https://github.com/user-attachments/assets/7cc91551-c806-4a02-8025-0affa744e835



## Install

Run these two commands inside Claude Code:

```
/plugin marketplace add thealchen/starcraft-sounds
/plugin install starcraft-sounds@starcraft-sounds
```

## Uninstall

```
/plugin uninstall starcraft-sounds@starcraft-sounds
```
